üåç Why ASTERIX CAT62 Matters

    ASTERIX (All Purpose Structured EUROCONTROL Surveillance Information Exchange) is the global standard for surveillance data exchange in Air Traffic Management (ATM).

CAT62 specifically carries:
    Processed system track data
    Aircraft position
    Velocity
    Track status
    Identification
    Flight plan correlation data

Unlike raw radar plots (CAT48), CAT62 represents consolidated, processed track information generated by Surveillance Data Processing Systems (SDPS).
  It is used in:
    Radar tracker systems (e.g., ARTAS)
    ATC automation
    Air situation displays
    Multisensor fusion environments

ADS-B + radar integration systems

CAT62 represents the authoritative operational air traffic picture.
